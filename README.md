# NOVA AI Assistant

NOVA is a futuristic AI assistant built with React, Express, OpenAI APIs, and modern UI technologies.

---

## Features

- AI chatbot
- Persistent chat memory
- Voice input
- Voice output
- Futuristic animated UI
- 3D animated orb
- Particle background
- Theme switching
- Responsive design
- File upload support
- Live web search
- Smooth animations

---

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Framer Motion
- React Three Fiber
- Drei

### Backend
- Node.js
- Express
- OpenAI API
- Multer

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```

---

## Environment Variables

Create `.env` inside server:

```env
OPENAI_API_KEY=your_api_key
```
